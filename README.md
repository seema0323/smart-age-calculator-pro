# Smart Age Calculator Pro

A modern age calculator built using HTML, CSS and JavaScript.

## Features

- Exact Age Calculation
- Total Days, Weeks, Hours and Minutes
- Birthday Countdown
- Zodiac Sign
- Chinese Zodiac
- Birth Day Detection
- Age Category
- Life Progress Percentage
- Dark / Light Mode
- Local Storage Support
- Copy Result Feature
- Responsive Design

## Tech Stack

- HTML5
- CSS3
- JavaScript
- GitHub
- Vercel
